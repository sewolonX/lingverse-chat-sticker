## 功能特性

- 🎨 **暗色/亮色模式** — 自动跟随系统主题，CSS 变量全覆盖
- 🖼️ **图片贴纸** — 支持任意 HTTP/HTTPS 图片链接，含 GM_xmlhttpRequest 回退加载
- 🎬 **视频贴纸** — 支持 mp4/webm/ogg 等常见视频格式
- 🔊 **音频贴纸** — 支持 mp3/wav/flac/aac 等常见音频格式
- ✨ **文字/Emoji 贴纸** — 纯文本或 emoji 直接插入聊天框
- 📋 **贴纸管理** — 添加、删除、导出、导入贴纸
- 🔍 **贴纸详情弹窗** — 点击聊天中的贴纸查看大图/详情，支持缩放与拖拽
- 📐 **贴纸大小调节** — 滑块调节聊天中贴纸的显示大小（5%–100%）
- 📦 **批量导入** — 支持 JSON 格式和 owo 表情包格式导入
- 🔗 **跨玩家可见** — 通过 `[E:...]` 标记协议，安装本脚本的玩家自动解析贴纸
- 🔒 **Base64 编码** — URL 经 base64 编码嵌入消息，避免特殊字符问题

## 使用教程

### 打开贴纸面板

安装脚本后，进入 [灵界游戏页面](https://ling.muge.info/game.html)，在聊天输入框左侧会出现一个 **贴纸按钮**（图片图标）：

![贴纸按钮示意]()

点击贴纸按钮，会在按钮上方弹出贴纸选择面板：

```
┌──────────────────────────┐
│  🖼️1  🖼️2  🖼️3  🖼️4  │
│  😊   🎵   📄   🖼️5  │
│  ...                     │
├──────────────────────────┤
│        [ 管理 ]          │
└──────────────────────────┘
```

- 面板以 4 列网格展示你的所有贴纸
- 再次点击贴纸按钮或点击面板外区域，面板关闭

### 发送贴纸

**图片/视频/音频贴纸**：在贴纸面板中直接点击贴纸缩略图，贴纸会自动插入聊天输入框并立即发送。

**文字/Emoji 贴纸**：点击后文字会被插入到输入框中，但**不会自动发送**，你可以继续编辑文字后再手动发送。

> 💡 贴纸通过 `[E:base64编码:名称]` 格式嵌入聊天消息中。安装了本脚本的玩家会自动将其渲染为图片/视频/音频，未安装的玩家则会看到原始的标记文本。

### 管理贴纸

点击贴纸面板底部的 **「管理」** 按钮，展开管理面板。

#### 添加贴纸

1. 在 **「名称」** 输入框中填写贴纸名称（作为展示和提示文字）
2. 在 **「URL」** 输入框中填写贴纸地址
   - 图片：直接粘贴图片 URL（如 `https://example.com/cat.png`）
   - 视频：粘贴视频 URL（如 `https://example.com/clip.mp4`）
   - 音频：粘贴音频 URL（如 `https://example.com/bgm.mp3`）
   - 文字/Emoji：直接输入文内容（如 `😊` 或 `www`）
3. 点击 **「添加」** 按钮

> ⚠️ 名称中不能包含 `: [ ]` 字符。

#### 删除贴纸

在管理面板的贴纸列表中，每个贴纸右上角有一个红色的 **×** 按钮，点击即可删除。

你也可以点击贴纸进入详情弹窗，然后点击 **「删除」** 按钮。

#### 导出贴纸

1. 展开管理面板
2. 点击 **「导出」** 按钮
3. 浏览器会自动下载一个 `lingverse-stickers-YYYY-MM-DD-HHMMSS.json` 文件

导出的 JSON 文件包含你当前的所有贴纸数据。

#### 导入贴纸

**导入本脚本导出的 JSON：**

1. 展开管理面板
2. 点击 **「导入」** 按钮（按钮文字变为「确认导入」）
3. 将 JSON 内容粘贴到输入框中
4. 点击 **「确认导入」** 或按 `Enter`

**导入 owo 表情包格式：**

本脚本也支持导入 [owo](https://github.com/zzyss86/emoticon) 格式的表情包 JSON（结构为 `{"分类名": {"type": "emoticon|emoji|image", "container": [...]}}`），直接粘贴即可自动识别转换。

> 💡 导入时会自动**去重**（相同 id 或相同 URL 的贴纸不会重复添加），已有的贴纸会被跳过。

### 查看贴纸详情

**在管理面板中**：点击任一贴纸缩略图，弹出详情弹窗。

**在聊天消息中**：直接点击已渲染的贴纸图片、视频按钮或音频按钮，弹出详情弹窗。

详情弹窗功能：

- 📋 查看贴纸名称、URL、原始数据
- 🔍 **缩放** — 滑块调节、鼠标滚轮、`−`/`+` 按钮（20%–500%）
- ✋ **拖拽平移** — 放大后可鼠标拖拽查看图片细节
- 📱 **触屏支持** — 双指缩放 + 单指拖拽
- 📨 **插入到聊天框** — 将当前贴纸发送到聊天
- 🗑️ **删除** — 从列表中移除该贴纸（仅自己的贴纸）
- 🔗 **在新标签页打开** — 直接打开远程 URL
- ➕ **添加到我的贴纸** — 看到别人发的贴纸时，可以一键添加到自己的列表

### 调整贴纸大小

在管理面板中找到 **「贴纸大小」** 滑块，拖动调节（5%–100%，默认 55%）。

- 百分比相对于聊天消息区域宽度计算
- 调整后立即生效，聊天中已显示的贴纸大小会实时更新
- 设置会自动保存在浏览器中，下次打开页面依然有效

## 贴纸类型

### 图片贴纸

图片贴纸在聊天中会被渲染为 `<img>` 标签，显示为可点击的缩略图。

**支持的格式**：png、jpg/jpeg、gif（包括动图）、webp、svg、bmp、ico、avif、tiff

**加载机制**：先尝试直接加载，加载失败时自动通过 `GM_xmlhttpRequest` 代理获取 blob 数据并回退渲染。这可以绕过部分 CDN 的跨域或 PNA（Private Network Access）限制。

**链接识别规则**：
- `http://` 或 `https://` 开头且不匹配视频/音频规则 → 视为图片
- 含有 `png/jpg/gif/webp/svg` 等图片扩展名

### 视频贴纸

视频贴纸在聊天中显示为一个蓝色的 **「▶ 名称」** 按钮。点击按钮后会在新标签页中打开视频。

> ⚠️ 视频无法在聊天框中内联播放（受聊天区域宽度限制），需要在新标签页中查看。

**支持的格式**：mp4、webm、ogv、mov、mkv、m4v、3gp、avi

**链接识别规则**：
- URL 中包含 `.mp4/webm/ogv/mov` 等视频扩展名
- URL 路径中含有 `/video/`
- URL 参数中含有 `video_id/vid/video/play=`

### 音频贴纸

音频贴纸在聊天中显示为一个白色的 **「🔊 名称」** 按钮。点击按钮后会在新标签页中打开音频进行播放。

> ⚠️ 音频同样需要在新标签页中播放。

**支持的格式**：mp3、wav、flac、aac、m4a、oga、opus、weba、wma、ogg

**链接识别规则**：
- URL 中包含 `.mp3/wav/flac/aac` 等音频扩展名
- URL 路径中含有 `/audio/` 或 `/music/`
- URL 参数中含有 `audio_id/music_id/song_id/track_id=`

### 文字/Emoji 贴纸

任何不以 `http://`、`https://`、`file://`、`ftp://` 开头，且不以 `/`、盘符（如 `C:\`）、网络路径（如 `\\`）开头的内容，都会被识别为文字贴纸。

**纯 Emoji 贴纸**（1–4 个 emoji 字符）会以更大字号（22px）显示在贴纸面板中。

文字贴纸点击后**不会自动发送**，而是追加到输入框末尾，方便你继续编辑。

## 工作原理

本脚本通过以下机制实现跨玩家的贴纸互传：

```
┌──────────┐     ┌──────────────────┐     ┌──────────┐
│ 玩家 A   │ ──→ │ 聊天消息          │ ──→ │ 玩家 B   │
│ 点击贴纸  │     │ [E:base64:名称]  │     │ 自动解析  │
└──────────┘     └──────────────────┘     └──────────┘
```

1. **发送侧**：点击贴纸时，脚本将贴纸 URL 进行 base64 编码，组装为 `[E:编码后URL:名称]` 格式的文本，写入聊天输入框并调用 `sendChat()` 发送
2. **消息格式**：`[E:` + payload + `:` + name + `]`，其中 payload 既可以是 base64 编码的 URL，也可以是原始 URL（当编码后超过 200 字限制时自动降级）
3. **接收侧**：脚本轮询和 MutationObserver 双重监控新消息，用正则 `\[E:([\s\S]+?):([^:\]]*)\]` 匹配并替换为对应的 DOM 元素
4. **数据存储**：所有贴纸数据存储于浏览器 `localStorage` 中，不会上传到任何服务器

**隐私说明**：贴纸数据完全保存在你的浏览器本地，导出/导入通过 JSON 文件在本地完成。唯一的网络请求是加载贴纸图片/视频/音频本身。

## 常见问题

<details>
<summary><strong>Q: 我发的贴纸别人看不到？</strong></summary>

对方也需要安装本脚本才能看到贴纸渲染效果。未安装脚本的玩家会在聊天中看到原始的 `[E:...:名称]` 文本。请邀请对方也安装本脚本。
</details>

<details>
<summary><strong>Q: 图片加载失败/显示"图片失效"？</strong></summary>

脚本会先尝试直接加载图片。如果失败（例如 CDN 跨域限制），会自动通过 GM_xmlhttpRequest 代理获取 blob 数据。如果仍然失败，请检查：

- 图片 URL 是否可直接访问（在浏览器新标签页中打开试试）
- URL 是否被防火墙或 CDN 防盗链阻止
- 脚本的 `@connect` 权限是否覆盖了图片域名（默认 `@connect *`）

</details>

<details>
<summary><strong>Q: 贴纸消息超过 200 字限制？</strong></summary>

灵界聊天消息有 200 字限制。当贴纸 URL 过长导致编码后超限时，脚本会：

1. 先尝试 base64 编码，如果超限则降级为原始 URL
2. 如果原始 URL 也超限，仍使用 base64（至少更安全）
3. 建议使用短链接或 CDN 缩短图片 URL

</details>

<details>
<summary><strong>Q: 如何和好友分享贴纸列表？</strong></summary>

在管理面板中点击 **「导出」** 下载 JSON 文件，将文件发送给好友。好友在管理面板中点击 **「导入」** → 粘贴 JSON 内容 → 确认导入。

</details>

<details>
<summary><strong>Q: 管理面板关不掉？</strong></summary>

当管理面板（管理模式）展开时，直接点击面板外部会提示「请先点击「退出管理」再关闭面板」。这是为了防止误关导致添加/编辑中的内容丢失。请点击底部的 **「退出管理」** 按钮收起管理面板后，再点击面板外部关闭。
</details>

<details>
<summary><strong>Q: 脚本更新后怎么升级？</strong></summary>

Tampermonkey 会按照脚本头部的 `@updateURL` 自动检查更新。你也可以手动检查：点击 Tampermonkey 图标 → 管理面板 → 检查更新。
</details>
